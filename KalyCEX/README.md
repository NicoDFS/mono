# Kaly Exchange Install Script  


## Main features.

-   Custodial wallet supporting KLC, BTC, ETH, BNB, TRX, USDT (KRC-20, ERC-20, BEP-20, TRC-20). Deposit and withdrawal.
-   Order matching engine. Trading pairs KLC-USDT, BTC-USDT, ETH-USDT, BNB-USDT, TRX-USDT.
-   Professional exchange interface for placing orders.    
-   Simplified interface for quick swap.
-   KYT transaction verification with Scorechain.
-   KYC verification of users with Sumsub.
-   SMS 2fa for users with Twilio.
